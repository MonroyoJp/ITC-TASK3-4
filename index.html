1. External GPU

An eGPU dock is an enclosure that you connect to your laptop or mini-PC (or sometimes a desktop) via a high-bandwidth interface (Thunderbolt, USB4, or similar). Inside the dock is a full desktop-class graphics card (GPU) plus power supply, cooling, ports, etc.

How it improves functionality

Better Graphics in general ▼

If your laptop has a weak or integrated GPU, connecting an eGPU lets it run applications (games, 3D modelling, rendering) that rely heavily on graphics for much more smoother performance. The external GPU takes over the heavy lifting.

Faster Processing ▼

For video editors, 3D artists, or those using AI / deep learning, more GPU power means faster rendering, smoother previews, better ability to handle high resolution, effects, etc.

Portability + Docking ▼

You can use the laptop independently (for light tasks), but when you need GPU power you attach the eGPU dock. Also many eGPU boxes double as docking stations: extra USB ports, Ethernet, displays etc. The AI Box from Gigabyte is one such example.

Upgradability ▼

When a new GPU comes out, you can often just replace the graphics card inside the dock rather than buying a new whole laptop.

2. Drawing Tablet

Drawing tablets are peripherals that let you draw / sketch / annotate directly into a computer using a stylus pen.

They come in two main types: Pen tablets without built-in screen (you draw on the tablet surface, see result on the monitor).
Pen displays (the tablet surface is a screen) so you draw directly where you see.

Recent models (like those ranked in “best drawing tablets 2025”) emphasize higher resolution, responsive stylus input (pressure sensitivity, tilt, low latency), better color accuracy, built-in shortcut keys/dials, etc.

How it improves functionality

Digital art / illustration / design ▼

More precise control (pressure sensitivity, tilt), better responsiveness = more natural drawing.

Photo editing / Fine retouching ▼

Stylus lets you do fine brush strokes, masking, detailed edits more accurately than with a mouse.

Annotation / Mark-ups ▼

Designers, teachers, or professionals who annotate documents or PDFs can do so more naturally.

Workflow / productivity ▼

Shortcut keys / dials speed up access to common tools (e.g. undo, brush size, zoom). Better display tablets reduce need to “guess” where you’re clicking or drawing.

Case Study: Explain what occurs within the computer when you hit a key on the keyboard and the letter shows up on the screen. Sort it by intake, processing, and output.

Intake

Our keyboard uses a switch matrix which composes of a grid of intersecting row and column wires. Each key sits at one intersection, forming an open circuit by default. When you press a key, it closes the circuit between its row and column, allowing a small electric current to pass. The keyboard's microcontroller scans all intersections rapidly, then detects the newly closed circuit, and identifies which key was pressed. It then sends that key's scan code to the computer through USB or Bluetooth.

Processing

The operating system receives the signal, converts the scan code into the correct character using the keyboard layout, and sends it to an active application.

Output

The application displays the corresponding letter on screen, triggering GPU and monitor updates so you see the character appear instantly.

Task 10

Write a comprehensive essay to explain why a computer cannot function without software on its own. Incorporate examples from everyday life, such as playing games, using applications, or browsing the web.

Why a Computer Cannot Function Without Software

A computer may appear to be a powerful and intelligent machine, but without software, it is merely an empty shell of electronic components. Hardware alone—such as the monitor, keyboard, mouse, CPU, and memory—cannot perform any meaningful task unless guided by software. Software serves as the instructional brain that tells the hardware what to do, how to do it, and when to do it. Without software, even the most advanced computer is lifeless and incapable of carrying out the simplest operations.

To understand this, consider what happens when you press the power button on your computer. The first thing that runs is a small piece of software called the BIOS or firmware. This program checks that all hardware components are working and then loads the operating system, such as Windows, macOS, or Linux. The operating system is itself software—it manages memory, handles input and output devices, organizes files, and allows the user to interact with the system through a graphical interface. Without it, your computer would only display a blank screen because the hardware would have no instructions to follow.

Software also makes computers useful in everyday life. For instance, when you play a video game, the game software tells the processor how to generate graphics, calculate movements, and respond to your keyboard or controller inputs. Without the game software, your computer would not know how to display characters, music, or gameplay interactions—it would simply sit idle. Similarly, when you open applications such as Microsoft Word, Spotify, or Photoshop, it is the software that translates your clicks and commands into meaningful actions.

The keyboard, mouse, and screen only serve as input and output tools; the real coordination happens through software instructions.

Browsing the web is another good example. When you open Google Chrome or another browser, software code communicates with servers across the internet to retrieve web pages, display images, and process videos. If there were no browser software, the hardware would not know how to interpret internet data or show it in a readable form. Even something as simple as watching a YouTube video requires layers of software: the browser to display it, the operating system to manage it, and various drivers to handle graphics and audio output.

From a broader perspective, software bridges the gap between human intent and machine operation. Humans think in terms of tasks—writing an essay, editing a photo, or sending a message—while computers operate in binary code. Software translates human commands into machine language that hardware can execute. Without this translator, humans could not communicate with computers effectively.

In conclusion, a computer without software is like a body without a mind—it has all the parts but no control or purpose. Software gives life, intelligence, and usability to computer hardware. Whether we are playing games, using applications, or browsing the web, every function we perform on a computer depends entirely on software guiding the hardware. Without it, computers would remain silent boxes of circuits, unable to serve the vital roles they play in our daily lives.
